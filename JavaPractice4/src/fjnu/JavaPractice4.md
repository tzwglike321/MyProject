# IOTest:
> * 基本IO操作
> * 从键盘接收字节流
> * 写入到当前目录下的src.txt文件中
> * 将src.txt文件内容复制到当前目录下dest.txt文件中
> * 将dest.txt文件内容显示到屏幕上

# FileInformation:
> * 用于存储文件信息

# GetFileTree:
> * 获取某个目录下的目录信息，目录从控制台输入
> * 目录信息包括该目录下的所有文件和文件夹的列表。对于每个文件夹显示其名称，修改日期。对于每个文件显示其名称，修改日期和大小
> * 目录的信息写出到文件中。注意，文件信息首先按类型分类，文件夹在前，文件在后，并且他们各自按照文件名字符串的顺序排序
