# implicit_view
> * 主界面，输入网址，点击浏览网页跳转浏览器浏览

# activity_browse
> * 自定义简单的webview

# MainActivity
> * 加载implicit_view，设置浏览网页的点击监听事件；
> * 点击浏览网页获取EditText的内容，调用隐式Intent启动
